# Original-Lizenztexte

In diesem Ordner werden die erforderlichen Original-Lizenztexte der tatsächlich ausgelieferten Drittanbieter-Komponenten gesammelt.

Vor einem Release muss geprüft werden:

- Welche Komponenten werden direkt mitgeliefert?
- Welche Tools werden automatisch installiert?
- Welche Lizenztexte müssen im Release-Paket enthalten sein?
- Stimmen `THIRD_PARTY_LICENSES.md`, README und Paketinhalt überein?
